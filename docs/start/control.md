# 控制台

## 注册控制台账号

登录[控制台](https://control.unit-one.top)，并且注册账号。

:::tip
资金所限，目前仅支持邮箱注册，如果你在邮箱中没有收到注册验证码的消息，麻烦去垃圾箱看一下，可能被系统当作骚扰邮件统一处理了。
:::

## 创建应用

进入应用管理子菜单项，点击+即可新建应用。目前仅支持微信小程序，后续会支持更多平台。填写名称、微信小程序的appid、微信小程序的appsecret，点击确认即可。注意这部分appid和appsecret是微信小程序的，需要你在微信小程序的控制台中获取。

创建成功后，列表中会新增你刚刚增加的应用，其中的appId和appSecret是我平台的，当然如果暂时不记住也没关系，后续是一直可查看的。

:::info
我平台的appId和appSecret在后续调用我提供的api时会用到。
:::
